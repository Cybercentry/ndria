## Ethereum public-goods track record

Direct contributions to Ethereum standards - published, open, and ecosystem-facing:

- **[ERC-8126: AI Agent Verification](https://eips.ethereum.org/EIPS/eip-8126)** - Leigh Cronian
  (@cybercentry), Chris Johnson. A framework for verifying AI agents registered via ERC-8004
  across five verification types (token, media, Solidity code, web app, wallet), using
  zero-knowledge proofs.
- **[ERC-8196: AI Agent Authenticated Wallet](https://eips.ethereum.org/EIPS/eip-8196)** - Leigh
  Cronian (@cybercentry). A standard for wallets that execute only with cryptographic proof of
  policy compliance - the execution layer of a modular trust framework (registration → ERC-8004,
  verification → ERC-8126, execution → ERC-8196).
